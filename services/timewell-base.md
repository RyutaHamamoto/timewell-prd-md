## 主要機能

### 1. SNS投稿自動生成

テーマを入力するだけで、各プラットフォームに最適化された投稿を**60秒で完成**：

- **Twitter/X**: 文字数制限を考慮した投稿
- **Instagram**: ハッシュタグ・画像提案付き
- **Facebook**: 長文・エンゲージメント重視

### 2. イベントページ自動生成

**ワンクリック**で以下をすべて自動生成：

- イベントバナー
- 説明文
- 申し込みフォーム
- リマインダーメール
- 参加者向け案内

### 3. オールインワンプラットフォーム

すべてが一つで完結：

| 機能 | 詳細 |
|------|------|
| SNS連携 | 複数プラットフォーム一括管理 |
| イベント管理 | 作成・集客・当日運営 |
| メンバー管理 | 属性管理・セグメント |
| 決済 | オンライン決済対応 |
| 分析 | エンゲージメント・ROI分析 |

### 4. AI分析機能

- メンバー行動データの自動分析
- 最適な投稿タイミングの提案
- 人気テーマの発見
- エンゲージメント向上施策の提案
