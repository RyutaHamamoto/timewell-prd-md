# Slide Customer

顧客向けカスタマイズスライド（表紙+課題スライド追加）の作成ワークフロー。

**参照スキル**:
- timewell-design-system（デザイン基盤）
- nanobanana-core（技術基盤）

---

## 用途

既存のホワイトペーパーに、特定顧客向けのスライドを追加する。

- 表紙のカスタマイズ（顧客名入り）
- 課題スライドのパーソナライズ（顧客の業界・課題に特化）
- 追加枚数: 2-3枚

---

## ワークフロー（6ステップ）

### Step 0: 日付確認と最新情報チェック

```markdown
1. Web検索で今日の日付を確認
2. 顧客企業の最新情報を確認
   - 企業HP
   - プレスリリース
   - 決算情報（上場企業の場合）
   - 業界ニュース
```

### Step 1: 顧客プロファイル作成

**特定の顧客向けに詳細に定義する。**

```markdown
# 顧客プロファイル: {企業名}

## 企業情報
- 企業名:
- 業種:
- 従業員数:
- 売上規模:
- 本社所在地:
- 上場区分:

## 担当者情報
- 氏名:
- 役職:
- 部門:
- 連絡先:

## 企業の課題（リサーチ結果）
- 公開情報から読み取れる課題:
  1.
  2.
  3.
- 業界共通の課題:
  1.
  2.

## 提案のポイント
- この企業に刺さるメッセージ:
- 避けるべき表現:
- 参考になる類似導入事例:

## カスタマイズ方針
- 表紙に入れる要素:
- 課題スライドの切り口:
```

### Step 2: 顧客企業リサーチ

```markdown
## Web検索項目
- "{企業名}" 課題
- "{企業名}" DX
- "{企業名}" 業務効率化
- "{業界}" トレンド 2024
- "{企業名}" ニュース

## 確認するソース
- 企業公式サイト
- IR資料（上場企業）
- 採用ページ（組織の課題が見える）
- SNS（企業公式、担当者）
```

### Step 3: カスタマイズスライド設計

```markdown
## カスタマイズ表紙

タイトル: {顧客名}様向け
        {サービス名}ご提案

サブタイトル: {顧客の課題}を解決する{サービス名}

日付: {提案日}
会社名: 株式会社TIMEWELL

---

## カスタマイズ課題スライド

ヘッダー: {顧客名}様の課題

ワンメッセージ:
{業界}における{具体的課題}で年間{数値}時間を損失

課題カード（Bento UI）:
1. {顧客固有の課題1}
2. {顧客固有の課題2}
3. {業界共通の課題}
4. {解決しないとどうなるか}
```

### Step 4: Nano Banana Pro生成

`nanobanana-core` を参照し、カスタマイズスライドを生成。

### Step 5: 既存PPTXに挿入

```javascript
// 既存のホワイトペーパーを読み込み
// 先頭にカスタマイズスライドを挿入
// または特定位置に挿入

const pptx = new pptxgen();

// カスタマイズ表紙
const coverSlide = pptx.addSlide();
coverSlide.addImage({ path: customCover, x: 0, y: 0, w: '100%', h: '100%' });

// カスタマイズ課題スライド
const problemSlide = pptx.addSlide();
problemSlide.addImage({ path: customProblem, x: 0, y: 0, w: '100%', h: '100%' });

// 既存スライドを追加（スライド2以降）
// ...
```

## 業界別課題テンプレート

### 製造業
- 技術文書の検索に時間がかかる
- 熟練者のノウハウが属人化
- 品質管理情報の横断検索が困難

### 金融業
- コンプライアンス関連文書の管理
- 過去の審査事例の検索
- 規制対応の情報収集

### IT企業
- 技術ドキュメントの更新・検索
- 新人エンジニアの育成効率
- ナレッジの共有と再利用

### 小売・流通
- 店舗マニュアルの検索
- 本部への問い合わせ対応
- 商品情報の横断検索
