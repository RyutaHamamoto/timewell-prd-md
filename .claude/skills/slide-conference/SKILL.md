# Slide Conference

講演・カンファレンス資料（20-50枚）の作成ワークフロー。

**参照スキル**:
- timewell-design-system（デザイン基盤）
- nanobanana-core（技術基盤）

---

## 用途

- カンファレンス登壇
- セミナー・ウェビナー
- 社外向けプレゼンテーション
- イベント講演

**特徴**: ストーリーテリング重視、聴衆を引き込む構成

---

## ワークフロー（8ステップ）

### Step 0: 日付確認とイベント情報確認

```markdown
1. Web検索で今日の日付を確認
2. イベント情報の確認
   - イベント名
   - 開催日時
   - 会場/オンライン
   - 参加者属性
   - 持ち時間
   - 他の登壇者
```

### Step 1: 想定聴衆の定義

```markdown
# 聴衆プロファイル

## イベント情報
- イベント名:
- 開催日:
- 持ち時間: 分
- 形式: 登壇 / パネル / ワークショップ

## 聴衆属性
- 参加者数:
- 職種・役職:
- 業界:
- 知識レベル:
- 参加目的:

## 聴衆の期待
- 知りたいこと:
- 持ち帰りたいもの:
- 行動変容:

## 講演のゴール
- メインメッセージ（1つ）:
- 記憶に残したい3つのポイント:
  1.
  2.
  3.
- 講演後のアクション:
```

### Step 2: 情報収集（Web検索）

```markdown
## 検索項目
- イベントの過去回の内容
- 他の登壇者のテーマ
- 聴衆の業界トレンド
- 最新の統計データ
- 引用可能な事例

## 講演テーマに関する最新情報
- 業界動向
- 技術トレンド
- 競合の動き
```

### Step 3: ストーリー設計

```markdown
# 講演ストーリー構成

## オープニング（10%）: 掴み
- フック（問いかけ/衝撃的事実/ストーリー）
- 自己紹介
- 本日のアジェンダ

## 導入（15%）: 課題提起
- 聴衆が共感する課題
- なぜ今このテーマか
- 解決しないとどうなるか

## メイン（50%）: 解決策と事例
- 解決策の提示
- 具体的な方法論
- 事例・実績
- デモ（あれば）

## まとめ（15%）: 行動喚起
- 3つのキーポイント
- 明日からできること
- リソース・参考情報

## クロージング（10%）: CTA
- Q&A
- 連絡先
- 次のステップ
```

### Step 4: スライドアウトライン作成

**30分講演の場合（約30枚）**:

```markdown
## オープニング（3枚）
1. タイトル
2. 自己紹介
3. アジェンダ

## 導入（5枚）
4. フック（衝撃的な数字/問い）
5. 課題①
6. 課題②
7. 課題の影響
8. 本日お伝えすること

## メイン（15枚）
9. ソリューション概要
10. ポイント①
11. ポイント①詳細
12. ポイント①事例
13. ポイント②
14. ポイント②詳細
15. ポイント②事例
16. ポイント③
17. ポイント③詳細
18. ポイント③事例
19. デモ/スクリーンショット
20. 導入効果
21. お客様の声
22. よくある質問
23. 比較/ポジショニング

## まとめ（5枚）
24. セクション区切り「まとめ」
25. キーポイント①
26. キーポイント②
27. キーポイント③
28. 明日からできること

## クロージング（2枚）
29. Q&A
30. CTA・連絡先
```

### Step 5: デザインガイドライン確認

`timewell-design-system` を参照。

**講演資料特有のポイント**:
- 文字は大きめ（遠くからも見える）
- 1スライド1メッセージ
- 視覚的なインパクト重視
- 「間」を作るスライド（シンプルな引用など）

### Step 6: Nano Banana Pro生成

`nanobanana-core` を参照。

**講演資料向けプロンプト追加指示**:
```
- Text must be large and readable from distance
- One key message per slide
- High visual impact
- Generous whitespace
```

### Step 7: スピーカーノート追加

```javascript
// PptxGenJSでスピーカーノートを追加
slide.addNotes('ここで〇〇について話す。例: ...');
```

### Step 8: リハーサル・時間調整

- 1枚あたり約1分が目安
- 時間超過の場合はスライド削減
- 重要でないスライドは補足資料へ

---

## セクション区切りスライド

講演では、セクション間に区切りスライドを入れる。

```
┌────────────────────────────────┐
│                                │
│                                │
│         ソリューション          │
│                                │
│                                │
└────────────────────────────────┘

- 背景: 薄いグラデーション
- テキスト: 大きく中央配置
- 聴衆が「次のセクションに入る」と認識
```

---

## 時間配分の目安

| 持ち時間 | スライド枚数 | 1枚あたり |
|---------|------------|----------|
| 15分 | 15-20枚 | 45秒-1分 |
| 30分 | 25-35枚 | 50秒-1分 |
| 45分 | 35-50枚 | 50秒-1分 |
| 60分 | 45-60枚 | 1分 |
